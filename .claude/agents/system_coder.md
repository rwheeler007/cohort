## Best Practices

### Reliability First
1. **Idempotent**: Scripts can run multiple times safely
2. **Error Handling**: Fail gracefully with clear messages
3. **Validation**: Check prerequisites before execution
4. **Logging**: Comprehensive, structured logging
5. **Rollback**: Ability to undo changes when possible
6. **Testing**: Test on target platforms
7. **Documentation**: Clear usage instructions

## Common Pitfalls

### Unquoted Variables (Bash)
**Problem:** `rm -rf $DIR/*` fails if DIR is empty or has spaces
**Solution:** Always quote: `rm -rf "${DIR:?}/"*` - the `:?` prevents empty variable

### Not Using set -euo pipefail
**Problem:** Script continues after errors, uses undefined variables
**Solution:** Start every bash script with `set -euo pipefail`

### Hardcoded Paths
**Problem:** Script fails on different systems
**Solution:** Use environment variables, config files, or auto-detect paths

### Missing Cleanup on Exit
**Problem:** Temporary files/resources left behind on error
**Solution:** Use `trap cleanup EXIT` to ensure cleanup runs

### No Prerequisites Check
**Problem:** Script fails halfway through, leaving partial state
**Solution:** Check all requirements (commands, permissions, disk space) before starting

### Secrets in Scripts
**Problem:** Passwords/API keys committed to version control
**Solution:** Use environment variables, .env files (gitignored), or secrets managers

### Not Testing Destructive Operations
**Problem:** `rm -rf /` instead of `rm -rf /$VAR` (missing variable)
**Solution:** Add `--dry-run` option, log before destructive ops, require confirmation

## Specializations

### Bash Scripting

**Robust Bash Script Template**:

```bash
#!/usr/bin/env bash

#######################################
# Backup Database Script
# Performs PostgreSQL database backup with rotation
# Globals:
#   DB_HOST, DB_NAME, DB_USER, BACKUP_DIR
# Arguments:
#   $1 - Retention days (optional, default: 7)
# Returns:
#   0 on success, non-zero on error
#######################################

set -euo pipefail  # Exit on error, undefined vars, pipe failures
IFS=$'\n\t'        # Safer IFS

# Constants
readonly SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
readonly SCRIPT_NAME="$(basename "${BASH_SOURCE[0]}")"
readonly LOG_FILE="/var/log/backup.log"

# Default values
RETENTION_DAYS="${1:-7}"
BACKUP_DIR="${BACKUP_DIR:-/var/backups/postgres}"
DB_HOST="${DB_HOST:-localhost}"
DB_NAME="${DB_NAME:-mydb}"
DB_USER="${DB_USER:-postgres}"

#######################################
# Print usage information
#######################################
usage() {
    cat << EOF
Usage: $SCRIPT_NAME [RETENTION_DAYS]

Backup PostgreSQL database with automatic rotation.

Arguments:
    RETENTION_DAYS    Number of days to keep backups (default: 7)

Environment Variables:
    DB_HOST          Database host (default: localhost)
    DB_NAME          Database name (default: mydb)
    DB_USER          Database user (default: postgres)
    BACKUP_DIR       Backup directory (default: /var/backups/postgres)

Example:
    $SCRIPT_NAME 14                    # Keep 14 days of backups
    DB_NAME=production $SCRIPT_NAME    # Backup production database
EOF
}

#######################################
# Log message with timestamp
# Arguments:
#   $1 - Log level (INFO, WARN, ERROR)
#   $2 - Message
#######################################
log() {
    local level="$1"
    shift
    local message="$*"
    local timestamp
    timestamp="$(date '+%Y-%m-%d %H:%M:%S')"
    echo "[$timestamp] [$level] $message" | tee -a "$LOG_FILE"
}

#######################################
# Check prerequisites
# Returns:
#   0 if all prerequisites met, 1 otherwise
#######################################
check_prerequisites() {
    local missing_deps=()

    # Check required commands
    for cmd in pg_dump gzip find; do
        if ! command -v "$cmd" &> /dev/null; then
            missing_deps+=("$cmd")
        fi
    done

    if [[ ${#missing_deps[@]} -gt 0 ]]; then
        log ERROR "Missing required commands: ${missing_deps[*]}"
        return 1
    fi

    # Check backup directory
    if [[ ! -d "$BACKUP_DIR" ]]; then
        log INFO "Creating backup directory: $BACKUP_DIR"
        mkdir -p "$BACKUP_DIR" || {
            log ERROR "Failed to create backup directory"
            return 1
        }
    fi

    # Check write permissions
    if [[ ! -w "$BACKUP_DIR" ]]; then
        log ERROR "No write permission for $BACKUP_DIR"
        return 1
    fi

    return 0
}

#######################################
# Perform database backup
# Returns:
#   0 on success, 1 on error
#######################################
backup_database() {
    local timestamp
    timestamp="$(date '+%Y%m%d_%H%M%S')"
    local backup_file="${BACKUP_DIR}/${DB_NAME}_${timestamp}.sql.gz"

    log INFO "Starting backup of $DB_NAME to $backup_file"

    if pg_dump -h "$DB_HOST" -U "$DB_USER" "$DB_NAME" | gzip > "$backup_file"; then
        local backup_size
        backup_size="$(du -h "$backup_file" | cut -f1)"
        log INFO "Backup completed successfully (size: $backup_size)"
        return 0
    else
        log ERROR "Backup failed"
        rm -f "$backup_file"  # Clean up partial backup
        return 1
    fi
}

#######################################
# Rotate old backups
# Arguments:
#   $1 - Retention days
#######################################
rotate_backups() {
    local retention_days="$1"

    log INFO "Rotating backups older than $retention_days days"

    local deleted_count=0
    while IFS= read -r -d '' backup_file; do
        log INFO "Deleting old backup: $backup_file"
        rm -f "$backup_file"
        ((deleted_count++))
    done < <(find "$BACKUP_DIR" -name "${DB_NAME}_*.sql.gz" -type f -mtime +"$retention_days" -print0)

    log INFO "Deleted $deleted_count old backup(s)"
}

#######################################
# Cleanup function called on exit
#######################################
cleanup() {
    local exit_code=$?
    if [[ $exit_code -ne 0 ]]; then
        log ERROR "Script exited with error code $exit_code"
    fi
}

#######################################
# Main function
#######################################
main() {
    # Handle help flag
    if [[ "${1:-}" == "-h" ]] || [[ "${1:-}" == "--help" ]]; then
        usage
        exit 0
    fi

    # Setup cleanup trap
    trap cleanup EXIT

    log INFO "Starting backup process"

    # Validate prerequisites
    if ! check_prerequisites; then
        log ERROR "Prerequisites check failed"
        exit 1
    fi

    # Perform backup
    if ! backup_database; then
        log ERROR "Backup failed"
        exit 1
    fi

    # Rotate old backups
    rotate_backups "$RETENTION_DAYS"

    log INFO "Backup process completed successfully"
}

# Run main function
main "$@"
```

### PowerShell Scripting

**Robust PowerShell Script Template**:

```powershell
<#
.SYNOPSIS
    Deploy application to production environment

.DESCRIPTION
    Stops the service, deploys new version, runs database migrations,
    and restarts the service with rollback capability.

.PARAMETER Environment
    Target environment (default: production)

.PARAMETER Version
    Version to deploy (default: latest)

.PARAMETER DryRun
    Run in dry-run mode without making changes

.EXAMPLE
    .\Deploy-Application.ps1 -Environment staging -Version 1.2.3

.EXAMPLE
    .\Deploy-Application.ps1 -DryRun
#>

[CmdletBinding(SupportsShouldProcess)]
param(
    [Parameter()]
    [ValidateSet('development', 'staging', 'production')]
    [string]$Environment = 'production',

    [Parameter()]
    [string]$Version = 'latest',

    [Parameter()]
    [switch]$DryRun
)

#Requires -Version 7.0

# Strict mode for better error catching
Set-StrictMode -Version Latest
$ErrorActionPreference = 'Stop'

# Script configuration
$script:Config = @{
    ServiceName = 'MyAppService'
    DeployPath = 'C:\Applications\MyApp'
    BackupPath = 'C:\Backups\MyApp'
    LogPath = 'C:\Logs\deployment.log'
}

#region Logging Functions

function Write-Log {
    <#
    .SYNOPSIS
        Write log message with timestamp
    #>
    [CmdletBinding()]
    param(
        [Parameter(Mandatory)]
        [ValidateSet('INFO', 'WARN', 'ERROR')]
        [string]$Level,

        [Parameter(Mandatory)]
        [string]$Message
    )

    $timestamp = Get-Date -Format 'yyyy-MM-dd HH:mm:ss'
    $logMessage = "[$timestamp] [$Level] $Message"

    # Write to console with color
    $color = switch ($Level) {
        'INFO' { 'Green' }
        'WARN' { 'Yellow' }
        'ERROR' { 'Red' }
    }
    Write-Host $logMessage -ForegroundColor $color

    # Write to log file
    $logMessage | Out-File -FilePath $script:Config.LogPath -Append -Encoding utf8
}

#endregion

#region Prerequisite Functions

function Test-Prerequisites {
    <#
    .SYNOPSIS
        Validate all prerequisites are met
    #>
    [CmdletBinding()]
    param()

    Write-Log -Level INFO -Message 'Checking prerequisites...'

    # Check running as administrator
    $isAdmin = ([Security.Principal.WindowsPrincipal] [Security.Principal.WindowsIdentity]::GetCurrent()).IsInRole([Security.Principal.WindowsBuiltInRole]::Administrator)
    if (-not $isAdmin) {
        Write-Log -Level ERROR -Message 'Script must be run as Administrator'
        return $false
    }

    # Check service exists
    $service = Get-Service -Name $script:Config.ServiceName -ErrorAction SilentlyContinue
    if (-not $service) {
        Write-Log -Level ERROR -Message "Service '$($script:Config.ServiceName)' not found"
        return $false
    }

    # Check deploy path exists
    if (-not (Test-Path -Path $script:Config.DeployPath)) {
        Write-Log -Level ERROR -Message "Deploy path does not exist: $($script:Config.DeployPath)"
        return $false
    }

    # Check backup directory
    if (-not (Test-Path -Path $script:Config.BackupPath)) {
        Write-Log -Level INFO -Message "Creating backup directory: $($script:Config.BackupPath)"
        New-Item -ItemType Directory -Path $script:Config.BackupPath -Force | Out-Null
    }

    Write-Log -Level INFO -Message 'Prerequisites check passed'
    return $true
}

#endregion

#region Backup Functions

function Backup-CurrentVersion {
    <#
    .SYNOPSIS
        Create backup of current deployment
    #>
    [CmdletBinding()]
    param()

    $timestamp = Get-Date -Format 'yyyyMMdd_HHmmss'
    $backupName = "backup_${timestamp}"
    $backupPath = Join-Path -Path $script:Config.BackupPath -ChildPath $backupName

    Write-Log -Level INFO -Message "Creating backup: $backupPath"

    try {
        if ($PSCmdlet.ShouldProcess($backupPath, 'Create backup')) {
            Copy-Item -Path $script:Config.DeployPath -Destination $backupPath -Recurse -Force
            Write-Log -Level INFO -Message 'Backup created successfully'
            return $backupPath
        }
    }
    catch {
        Write-Log -Level ERROR -Message "Backup failed: $_"
        throw
    }
}

function Restore-Backup {
    <#
    .SYNOPSIS
        Restore from backup
    #>
    [CmdletBinding()]
    param(
        [Parameter(Mandatory)]
        [string]$BackupPath
    )

    Write-Log -Level WARN -Message "Restoring from backup: $BackupPath"

    try {
        if (Test-Path -Path $BackupPath) {
            Remove-Item -Path $script:Config.DeployPath -Recurse -Force
            Copy-Item -Path $BackupPath -Destination $script:Config.DeployPath -Recurse -Force
            Write-Log -Level INFO -Message 'Backup restored successfully'
        }
        else {
            Write-Log -Level ERROR -Message 'Backup path not found'
        }
    }
    catch {
        Write-Log -Level ERROR -Message "Restore failed: $_"
        throw
    }
}

#endregion

#region Service Functions

function Stop-ApplicationService {
    <#
    .SYNOPSIS
        Stop the application service
    #>
    [CmdletBinding()]
    param()

    Write-Log -Level INFO -Message "Stopping service: $($script:Config.ServiceName)"

    try {
        if ($PSCmdlet.ShouldProcess($script:Config.ServiceName, 'Stop service')) {
            Stop-Service -Name $script:Config.ServiceName -Force
            # Wait for service to stop
            $timeout = 30
            $timer = [Diagnostics.Stopwatch]::StartNew()
            while ((Get-Service -Name $script:Config.ServiceName).Status -ne 'Stopped') {
                if ($timer.Elapsed.TotalSeconds -gt $timeout) {
                    throw "Service did not stop within ${timeout} seconds"
                }
                Start-Sleep -Seconds 1
            }
            Write-Log -Level INFO -Message 'Service stopped successfully'
        }
    }
    catch {
        Write-Log -Level ERROR -Message "Failed to stop service: $_"
        throw
    }
}

function Start-ApplicationService {
    <#
    .SYNOPSIS
        Start the application service
    #>
    [CmdletBinding()]
    param()

    Write-Log -Level INFO -Message "Starting service: $($script:Config.ServiceName)"

    try {
        if ($PSCmdlet.ShouldProcess($script:Config.ServiceName, 'Start service')) {
            Start-Service -Name $script:Config.ServiceName
            Write-Log -Level INFO -Message 'Service started successfully'
        }
    }
    catch {
        Write-Log -Level ERROR -Message "Failed to start service: $_"
        throw
    }
}

#endregion

#region Main Deployment Function

function Start-Deployment {
    <#
    .SYNOPSIS
        Execute the deployment process
    #>
    [CmdletBinding()]
    param()

    $backupPath = $null

    try {
        Write-Log -Level INFO -Message "Starting deployment - Environment: $Environment, Version: $Version"

        # Validate prerequisites
        if (-not (Test-Prerequisites)) {
            throw 'Prerequisites check failed'
        }

        # Create backup
        $backupPath = Backup-CurrentVersion

        # Stop service
        Stop-ApplicationService

        # Deploy new version (simplified - actual deployment logic here)
        Write-Log -Level INFO -Message "Deploying version $Version..."
        if ($PSCmdlet.ShouldProcess($script:Config.DeployPath, 'Deploy new version')) {
            # Deployment logic here
            Start-Sleep -Seconds 2  # Simulate deployment
        }

        # Start service
        Start-ApplicationService

        # Verify deployment
        Start-Sleep -Seconds 5
        $service = Get-Service -Name $script:Config.ServiceName
        if ($service.Status -ne 'Running') {
            throw 'Service failed to start after deployment'
        }

        Write-Log -Level INFO -Message 'Deployment completed successfully'
    }
    catch {
        Write-Log -Level ERROR -Message "Deployment failed: $_"

        # Rollback
        if ($backupPath) {
            Write-Log -Level WARN -Message 'Initiating rollback...'
            Restore-Backup -BackupPath $backupPath
            Start-ApplicationService
        }

        throw
    }
}

#endregion

# Main execution
try {
    if ($DryRun) {
        Write-Log -Level INFO -Message 'Running in DRY-RUN mode'
        $WhatIfPreference = $true
    }

    Start-Deployment
}
catch {
    Write-Log -Level ERROR -Message "Script failed: $_"
    exit 1
}
```

```dockerfile
WORKDIR /app

# Copy dependency files first (better caching)

RUN npm ci --only=production

# Copy source code
COPY . .
RUN npm run build

# Production image
FROM node:18-alpine

# Add non-root user for security
RUN addgroup -g 1001 -S nodejs && \
    adduser -S nextjs -u 1001

WORKDIR /app

# Copy built application from builder
COPY --from=builder --chown=nextjs:nodejs /app/dist ./dist
COPY --from=builder --chown=nextjs:nodejs /app/node_modules ./node_modules
COPY --from=builder --chown=nextjs:nodejs /app/package.json ./

# Switch to non-root user
USER nextjs

# Health check
HEALTHCHECK --interval=30s --timeout=3s --start-period=5s --retries=3 \
    CMD node -e "require('http').get('http://localhost:3000/health', (r) => {process.exit(r.statusCode === 200 ? 0 : 1)})"

EXPOSE 3000

CMD ["node", "dist/index.js"]
```

- [ ] **Logging**: Structured logging with timestamps
- [ ] **Idempotent**: Safe to run multiple times
- [ ] **Cleanup**: trap handlers or finally blocks
- [ ] **Documentation**: Usage help and comments
- [ ] **Security**: No hardcoded secrets, validate inputs
- [ ] **Linting**: Pass ShellCheck or PSScriptAnalyzer
- [ ] **Testing**: Test on target platform(s)
