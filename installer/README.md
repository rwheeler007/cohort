# Cohort Windows Installer

Builds a Windows .exe installer that bundles Python + Cohort + dependencies.
Business users download this, double-click, and get a working local AI agent
orchestration system without touching a terminal.

## Architecture

```
CohortSetup.exe (Inno Setup)
  |
  +-- Installs to: C:\Users\{user}\AppData\Local\Programs\Cohort\
  |     +-- python\           (embedded Python 3.13, ~30MB)
  |     +-- python\Lib\site-packages\  (cohort + uvicorn + starlette + pystray + ...)
  |     +-- agents\           (agent configs + personas)
  |     +-- cohort-launch.bat (entry point)
  |     +-- cohort.ico
  |
  +-- Start Menu shortcut -> cohort-launch.bat
  +-- Desktop shortcut (optional)
  +-- Startup entry (optional)
```

## What happens on first launch

1. `cohort-launch.bat` runs with embedded Python (no console window)
2. `cohort launch` detects no `settings.json` -> first run
3. Server starts on `localhost:5100`, browser opens to setup wizard
4. Setup wizard (in browser):
   - Detects GPU/VRAM
   - Checks for Ollama, helps install if needed
   - Downloads best model for the hardware (~5-8GB)
   - Verifies inference works
   - Configures content feeds (optional)
5. Tray icon appears with "Open in Browser" / "Quit"

After first run, subsequent launches go straight to the dashboard.

## Building the installer

### Prerequisites

- Windows 10/11
- Python 3.11+ (build host only, not bundled)
- pip
- [Inno Setup 6.x](https://jrsoftware.org/isinfo.php) (free)
- Internet connection (downloads Python embed + packages)

### Steps

```bash
cd installer

# 1. Build a wheel first (from cohort root)
cd ..
python -m build
cd installer

# 2. Assemble the payload
python build_installer.py

# 3. Build the .exe (requires Inno Setup on PATH)
iscc cohort_installer.iss

# Output: installer/output/CohortSetup-0.3.0.exe
```

### Options

```bash
python build_installer.py --clean          # Clean rebuild
python build_installer.py --from-pypi      # Use PyPI instead of local wheel
python build_installer.py --skip-download  # Reuse cached downloads
```

## Payload size

| Component | Size |
|-----------|------|
| Embedded Python 3.13 | ~30 MB |
| Cohort + dependencies | ~40 MB |
| Agents (configs only) | ~2 MB |
| **Installer .exe** | **~80-100 MB** |
| Model (downloaded on first run) | ~5-8 GB |

## File structure

```
installer/
  cohort_installer.iss    -- Inno Setup script
  build_installer.py      -- Payload assembly script
  README.md               -- This file
  assets/
    cohort.ico            -- App icon (generated by build script)
  payload/
    cohort-launch.bat     -- Launcher (no console window)
    python/               -- Embedded Python (built)
    site-packages/        -- Installed packages (built)
    agents/               -- Agent configs (built)
  .cache/                 -- Downloaded archives (gitignored)
  output/                 -- Built installer .exe (gitignored)
```
