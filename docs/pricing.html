Frequently Asked Questions

What is the Agent Store?

The Agent Store is Cohort's agent distribution service. When you run Cohort, it connects to the Store to fetch agent configurations, prompts, and learned expertise. All 23 agents ship free at launch, with new specialists added regularly. All agents run on YOUR hardware using local inference -- the Store just delivers the intelligence.

What hardware do I need?

A single consumer GPU with 8-12GB VRAM runs the default model (qwen3.5:9b) comfortably. An RTX 3060/3070/3080 or equivalent is plenty. CPU-only mode works but is slower. No cloud GPU required.

Is the free tier actually usable in production?

Yes. All 23 agents ship free -- orchestration, marketing, content strategy, analytics, Python, web, JavaScript, security, QA, docs, code archaeology, and more. Plus compiled roundtables, executive briefings, and work queue. The same engine runs across all tiers. Pro adds premium pipelines like the Website Creator and Content Monitor when it launches.

Why are executive briefings free?

Executive briefings are how you see what your AI team accomplished. We want every Cohort user -- free or paid -- to start their day with a briefing. It's the best way to understand the value of multi-agent orchestration, and it naturally shows you what Pro and Enterprise capabilities could add to your workflow.

Can I use cloud models instead of local ones?

Yes, on all tiers. Cohort's Smartest tier runs locally by default (35B MoE on dual-GPU, 9B on single-GPU), but you can override any tier to use Claude, GPT, or any OpenAI-compatible API by adding your own API key in Settings. Built-in token budget limits prevent surprise bills.

What does "protocol-first" mean for vendor lock-in?

Cohort uses duck-typed Python protocols for agents, storage, and inference. Implement the interface and Cohort uses it. No base classes, no vendor SDK. pip install cohort has only two lightweight dependencies. If you stop paying, your agents, data, and workflows remain fully functional on the open source tier.

When does Pro launch?

The Agent Store and Pro tier are coming soon. Star the repo on GitHub to get notified. When they launch you'll be able to sign up for a free trial -- no credit card required.

Will you offer annual pricing?

Yes. Annual pricing details will be announced at launch. Enterprise pricing is negotiated per contract. Contact us for details.
